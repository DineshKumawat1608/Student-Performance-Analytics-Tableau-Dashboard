# 📊 Student Performance Tableau Dashboard

An interactive Tableau dashboard designed to help educators and school administrators monitor student performance, attendance, homework completion, and teacher-parent communication through a centralized reporting interface.

The dashboard transforms educational data into actionable insights using interactive visualizations, dynamic filters, and KPI-driven reporting.

---

# 📌 Dashboard Pages

## 1. Executive Overview

Provides a high-level summary of academic performance through interactive KPIs and visualizations.

### Highlights

- Total Students
- Total Subjects
- Average Exam Score
- Average Attendance
- Homework Completion Rate
- Attendance Status Distribution
- Homework Completion Distribution
- Subject-wise Performance
- Grade-wise Performance
- Student Demographics
- Monthly Trends

---

## 2. Student Analysis

Supports deeper academic analysis through interactive visualizations.

### Highlights

- Highest Performing Students
- Students Requiring Improvement
- Attendance by Subject
- Homework Completion vs Exam Score
- Average Age vs Exam Score
- Teacher-Parent Communication Analysis

---

## 3. Student Details

Provides student-level reporting for detailed academic monitoring.

### Highlights

- Student Profile
- Academic Summary
- Attendance Summary
- Parent Communication
- Student Rank
- Contact Information
- Dynamic Filters

---
